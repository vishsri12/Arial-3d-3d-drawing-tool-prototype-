# ✨ AetherPen 3D – Hand Gesture Drawing Studio

AetherPen 3D is a futuristic browser-based drawing application that allows users to create digital artwork using only hand gestures and a webcam.

Using Google MediaPipe's real-time hand tracking technology, users can draw, erase, and interact with a virtual canvas without touching a mouse, keyboard, or touchscreen.

---

## 🚀 Features

### 🎨 Gesture-Based Drawing
- Draw in the air using your index finger.
- Real-time stroke rendering.
- Smooth and natural drawing experience.

### 🧽 Gesture-Based Eraser
- Open-palm gesture activates eraser mode.
- Adjustable eraser size.
- Non-contact canvas editing.

### 🎯 Smart Motion Smoothing
- Reduces cursor jitter.
- Produces cleaner strokes.
- Improves overall user experience.

### 🌈 Custom Brush Controls
- Multiple neon color presets.
- Custom color picker.
- Adjustable brush thickness.

### 📊 Live Telemetry Dashboard
- Cursor coordinates tracking.
- Tracking confidence display.
- FPS monitoring.
- Drawing statistics.

### ↩️ Undo & Redo
- Reverse mistakes instantly.
- Restore previous actions.

### 💾 Export Artwork
- Save drawings as PNG images.
- Custom cyberpunk-themed export background.

### 📱 Responsive Design
- Works on desktop and laptop browsers.
- Adaptive interface for different screen sizes.

## 🧠 Gesture Controls

| Gesture | Action |
|----------|---------|
| ☝️ Index Finger Up | Draw |
| 🖐️ Open Palm | Erase |
| ✊ Fist / Neutral Hand | Hover Mode |

---

## 📸 How It Works

1. User grants webcam permission.
2. MediaPipe detects hand landmarks in real time.
3. Finger positions are converted into canvas coordinates.
4. Drawing and erasing gestures are recognized.
5. Artwork is rendered on a live canvas.
6. Users can export the final drawing as a PNG image.
